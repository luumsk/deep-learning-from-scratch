# Deep Learning From Scratch

A concept-first repository to understand deep learning from scratch, with code, visualization, and minimal experiments.


## Overview

This project focuses on how deep learning works internally, not just how to use frameworks.

Each concept is implemented in three layers:

	- Intuition (minimal examples)
	- From-scratch implementation (NumPy / pure Python)
	- Framework equivalent (PyTorch)

The goal is to build strong understanding of:

	- gradients and optimization
	- representation learning
	- architectural building blocks


## Learning Path

The repository follows a bottom-up progression:

**Phase 1 — Fundamentals**

    - Linear layer
    - Activation functions (ReLU, Sigmoid, Tanh, GELU)
    - Loss functions (MSE, Cross-Entropy)
    - Gradient descent
    - Backpropagation

**Phase 2 — Neural Networks**

    - MLP (multi-layer perceptron)
    - Weight initialization
    - Vanishing / exploding gradients
    - Regularization (dropout, weight decay)
    - Normalization (batch norm)

**Phase 3 — Vision (CNN)**

    - Convolution
    - Stride and padding
    - Pooling
    - Feature maps
    - Residual connections

**Phase 4 — Attention**

    - Embeddings
    - Query / Key / Value
    - Scaled dot-product attention
    - Multi-head attention
    - Masking
    - Positional encoding

**Phase 5 — Mini Transformer**

    - Transformer encoder block
    - Feedforward layer
    - Layer normalization
    - Residual connections
    - Tiny sequence task
